# IoT Voice Assistant with OLED & LED Indicators

This project implements a lightweight **Raspberry Pi–based Voice Assistant** using Python.  
It integrates:
- **Voice Recognition** for user queries  
- **Gemini AI API** for intelligent responses  
- **Text-to-Speech (TTS)** for spoken replies  
- **OLED Display** for real-time status updates  
- **LED Indicators** for system activity feedback  

---
---

## 📦 Requirements

| Component                     | Purpose                                   |
| ----------------------------- | ----------------------------------------- |
| Raspberry Pi 3B/4             | Main controller                           |
| USB Microphone                | Speech input                              |
| Speaker / 3.5mm Jack          | Voice output                              |
| OLED Display (SSD1306/SH1107) | Status display                            |
| LED (with resistor)           | Visual indicator for listening/responding |

 

# 🎙️ Jarvis Voice Assistant with OLED & LED Indicators  

This project implements **Jarvis**, a customizable **Raspberry Pi–based Voice Assistant** that integrates with **Google Gemini AI** through **LiveKit**. It features:  
- 🎛️ **LED indicators** to visually show when the system is listening or responding  
- 🖥️ **OLED display** for real-time text status updates  
- 🎤 **Voice input/output** for natural interaction  
- 🤖 **AI-powered responses** for queries and commands  

---

## ⚙️ Features
- 🎙️ Always listening voice assistant  
- 🟢 LED status indicators (listening / processing / responding)  
- 🖥️ OLED screen status updates  
- 🌐 Uses Google Gemini API via LiveKit  
- 🎵 Can be extended for media playback, IoT control, or automation  
- 🧩 Modular design for easy customization  

---

## 🌍 Applications
- 🏠 **Home Automation** → Control IoT devices via voice  
- 📚 **Education** → AI tutor with instant Q&A  
- 📊 **Office Assistant** → Notes, reminders, task management  
- ♿ **Accessibility** → Assistive tech for differently-abled users  
- 💡 **Prototyping & Research** → Build your own AI/IoT solutions  
- 🎮 **Entertainment** → Interactive voice-driven media hub  

---

## ✅ Pros
- 💡 Lightweight design, runs on Raspberry Pi 3B/4  
- ⚡ Real-time response with AI integration  
- 🎛️ Visual + Audio feedback (LED + OLED)  
- 🔌 Expandable to smart devices and IoT  
- 🛠️ Open-source & customizable  

---

## ⚠️ Cons
- 🌐 Requires internet for AI responses  
- 🔋 High power usage if running continuously  
- 🎤 Voice recognition may misinterpret in noisy environments  
- 🖥️ Limited by Raspberry Pi’s processing power
